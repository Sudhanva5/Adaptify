# Adaptify

**Idea to Ads in 2 minutes.**

Adaptify is an AI-powered creative generation platform that turns a plain-language campaign brief into on-brand ad creatives — then auto-adapts them to 16+ platform formats in one click.

Built by **Team Token Limit**.

## Features

- **Prompt-to-creative** — Describe your campaign in plain English, get 3 creative directions in ~60 seconds
- **Brand memory** — Learns from your past campaigns (CTR, CPL, conversions) to improve every new generation
- **3 variants per brief** — Pick the direction that fits, powered by performance data
- **16+ format adapts** — Instagram, Facebook, YouTube, LinkedIn, X/Twitter, TikTok, Pinterest, Snapchat, WhatsApp, Telegram, Threads and more

## Tech Stack

- Single-page HTML/CSS landing page
- Fonts: Source Serif 4 (display) + Plus Jakarta Sans (body)
- Icons: Phosphor Icons
- Deployment: Railway (Docker/nginx)

## Deploy to Railway

1. Push to GitHub
2. Connect the repo on [Railway](https://railway.app)
3. Railway auto-detects the Dockerfile and deploys

## Local Development

Just open `index.html` in your browser.
